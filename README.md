# Foodiebuddy

## Description

The idea for the app came from my girlfriend. The idea is to create an app where we could save gastronomy points of various types that we visited and rate them, thus creating a map of memories and places to return to.

The application was developed primarily for phones, so its layout in the desktop version at this point is virtually identical.

**Disclaimer**:
The app is currently blocked for third-party access for my personal use. I am in the process of redesigning it so that anyone can use it and create their own ranking :)

## Whats inside?

- Next.js
- React
- Prisma
- Mongodb
- Nextauth
- Typescript
- ReactLeaflet
